Class EventResource

EventResource provides access to Server-Sent Events (SSE) for real-time updates.

Example

// Subscribe to events
const subscription = await client.events.subscribe({
  onEvent: (event) => {
    console.log('Received event:', event.event, event.item);
  },
  onError: (error) => {
    console.error('SSE error:', error);
  }
});

// Later, close the connection
subscription.close();

subscribe

  • subscribe(options): Promise<SseSubscription>

  • Subscribe to real-time events via Server-Sent Events.

    This method establishes an SSE connection to receive real-time updates for entity changes (tasks, projects, teams, etc.).

    Note: This uses fetch with ReadableStream for broad compatibility. For browser environments, you may also use getStreamUrl() with EventSource.

    Parameters

    • options: SubscribeOptions

      Subscription options including event handlers

    Returns Promise<SseSubscription>

    A subscription object with close() method

    Example

    const subscription = await client.events.subscribe({
      onEvent: (event) => {
        switch (event.event) {
          case 'task.create':
            console.log('New task created:', event.item);
            break;
          case 'task.update':
            console.log('Task updated:', event.item);
            break;
        }
      },
      onConnected: (connectionId) => {
        console.log('Connected with ID:', connectionId);
      },
      onError: (error) => {
        console.error('Error:', error);
      },
      onClose: () => {
        console.log('Connection closed');
      }
    });

    // Close when done
    subscription.close();
